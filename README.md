# vim

* /searchTerm - finds searchTerm
* n for next occurence of the term
* zz to bring the current line to the middle of the screen
* so % - source the current file
* :bd close the buffer
* :e ~/.vimrc - edit 
* :pwd - show the current working directory
* :u - undo
* d - delete
* j - down
* k - up
* V - select the line
* v - enter visual mode
* y - yank
